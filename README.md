# 🚀 My Stack Helper API

Uma API robusta desenvolvida em **Node.js** que auxilia desenvolvedores a buscar informações técnicas de pacotes no ecossistema NPM e gerenciar uma lista personalizada de favoritos com persistência de dados.

## 📌 Sobre o Projeto

Este projeto foi criado com o objetivo de facilitar a curadoria de tecnologias para novos projetos. Ele se conecta à API oficial do NPMS para buscar metadados de bibliotecas e permite salvar os pacotes escolhidos em um arquivo físico JSON, simulando um banco de dados local.

### 🛠 Tecnologias Utilizadas

* **Node.js**: Ambiente de execução Javascript.
* **Express**: Framework web para criação de rotas e middlewares.
* **Axios**: Cliente HTTP para consumo da API externa (NPMS).
* **FS (File System)**: Módulo nativo do Node para manipulação de arquivos e persistência de dados.
